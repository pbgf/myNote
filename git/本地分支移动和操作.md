# 移动
* git checkout [hash] 移动HEAD到指定hash的点上
* git checkout HEAD^ 移动到HEAD的前一个结点
* git checkout HEAD~[num] 移动到HEAD的前num个结点
* git reset [hash][^][~num] 撤销变更到 某个结点
* git revert [hash] 保留当前结点新建一个结点 和 指定的结点一致<br>
